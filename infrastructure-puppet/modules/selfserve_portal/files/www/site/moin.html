Migrate a Moin wiki to Confluence

Important Note: Ticking the 'Import all revisions for all pages' checkbox will preserve and import page revisions for all pages. For Medium to Large moin wikis this could significantly add to the time it takes to migrate all pages to Confluence. (i.e could take 2 hours or more to complete.)

As an example: A 50 page wiki with an average 10 edits per page will add up to improrting 500 pages if you include history. Importing in this context includes the downloading of each page revision, the export of each page revision, the conversion of each page revision to Confluence format; and finally the upload of each page revision to our Confluence instance. So each migration step is multiplied

Note that we are absolutely fine with projects choosing to import history but we just wanted to make you aware of the possible significant increase in processing time should you choose to do so.
